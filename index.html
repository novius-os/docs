Novius OS Documentation

Version française

Novius OS is a new open-source CMS powered by the latest web technology, such as HTML5 and FuelPHP. Find out more at novius-os.org.

The documentation is at very early stage. Nevertheless, writing the documentation is an on-going process so you can expect frequent, yet incomplete, publications.
Do not hesitate to take part in the writing. After all, Novius OS is an open source and collaborative project.
