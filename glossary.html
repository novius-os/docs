Developers only ^ top
These terms are meant for developers only and won't be communicated to end users. They are not to be translated either.
App Engine   App Maker The applications' common base  
App Desk   mp3Grid, mp3Table, App Home, App Grid Standard UI for applications' home. It includes a main grid, surrounded by inspectors. Inspired by music players (e.g. iTunes, Banshee), it was codenamed 'mp3Grid'.  
Launcher   Icon, Home tab icon, Home tab application    
Enhancer   Add-on, Plug-in Functions to be added to a WYSIWYG (listed under Applications in the toolbar). Standard enhancers do not provide URLs (e.g. contact form).  
URL enhancer     Specific type of enhancers which provide URLs (e.g. blog).  
Sharer   Sharing provider, Connector, Micro-data Allows data sharing between applications and to/from external services (e.g. social media)  
Dispatcher   Connector Receives an event from an application and transmit it to the applications which are meant to react.  
Utility view     View unavaible to the end users but necessary for the application to run (e.g. Media Centre in a modal window).  
Locale     Language/country pair, e.g. en-gb (English, UK) or de-ch (German, Switzerland)  
Observer     Comes from FuelPHP. Adds pre/post processing to insert, update, save or delete.  
Behaviour     A behaviour is an observer. In addition, it allows to add new functions to an object. E.g. translatable, publishable.  
Tree grid     Layout widgets. Used for the views of the main grid, for instance.
