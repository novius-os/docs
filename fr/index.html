Documentation Novius OS

English version

Novius OS est un nouveau CMS open source motorisé par les dernières technologies web, comme HTML5 et FuelPHP. Découvrez-le sur novius-os.org.

La documentation en est à ses débuts. Néanmoins, la rédaction de la documentation étant en cours, les mises à jour sont fréquentes, même si encore incomplètes.
N'hésitez pas à prendre part à cette rédaction. En effet, Novius OS est un projet open source et collaboratif.
